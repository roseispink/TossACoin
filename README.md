# Projekt na przedmiot Technologie Handlu Elektronicznego
## Strona internetowa do zbiórek dla zwierząt


Projekt zakłada stworzenie w pełni funkcjonalnej strony internetowej umożliwiającej rejestrację, logowanie się, tworzenie zbiórek oraz wpłacanie na daną zbiórkę za pomocą zadanych metod płatności internetowych.
